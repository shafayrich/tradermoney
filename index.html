📊 Trading Logic & Signal Pipeline

The bot looks for EMA crossover events (fast EMA crosses above/below slow EMA). Every potential signal passes through multiple confirmation filters (all togglable). A signal only fires when all enabled indicators agree.

RSI (14)

Bullish: RSI ≥ 30 | Bearish: RSI ≤ 70

MACD

MACD line must agree with crossover direction.

VWAP

Price above VWAP for buys, below for sells.

Bollinger Bands

Price must be within the bands (lower*0.99 – upper*1.01).

ADX (14)

ADX ≥ 20 means a strong trend exists.

Volume Confirm

Volume > 1.5× the 20‑period average.

🏦 Broker Connection Guides

Select your broker from the app's sidebar and enter the required credentials. Below are step‑by‑step instructions for obtaining those credentials.

Alpaca

  1. Sign up at alpaca.markets.
  2. Go to Paper Trading (recommended) or Live Trading.
  3. Navigate to API and generate a new API key. Note the API Key ID and Secret Key.
  4. In TraderMoney, fill in API Key and Secret Key. Check Paper Trading if using paper account.

Interactive Brokers

  1. Install TWS or IB Gateway from IB website.
  2. Log in with your paper or live account.
  3. In TWS/Gateway: File → Global Configuration → API → Settings.
    ✅ Enable ActiveX and Socket Clients
    Port: 7497 (paper) / 7496 (live)
    Trusted IP: 127.0.0.1
    Uncheck Read‑Only API.
  4. In TraderMoney: Host = 127.0.0.1, Port = 7497 (paper) or 7496 (live), Client ID = 1.

Tradier

  1. Open an account at tradier.com.
  2. Go to API and create a new application. You'll receive an Access Token.
  3. Find your Account Number in the dashboard.
  4. In TraderMoney, enter the Access Token and Account ID.

Binance

  1. Create an account at binance.com.
  2. Go to API Management and create a new API key (enable Spot & Margin Trading).
  3. Copy the API Key and Secret Key.
  4. For paper trading, register at testnet.binance.vision and use testnet keys.
  5. In TraderMoney: enter the keys and check Testnet for paper trading.

Bybit

  1. Register at bybit.com.
  2. Go to API and create a new API key (enable Spot permissions).
  3. Copy the API Key and Secret.
  4. For testnet, sign up at testnet.bybit.com.
  5. In TraderMoney: enter key/secret and check Testnet if needed.

OKX

  1. Create an account at okx.com.
  2. Go to API and generate a new key. Set a Passphrase and enable Trade/Spot permissions.
  3. Copy API Key, Secret Key, and remember your Passphrase.
  4. For demo, use OKX Demo.
  5. In TraderMoney: fill in all three fields and check Demo Trading if using demo.
